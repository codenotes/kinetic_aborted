ModuleGather¶

class music21.test.commonTest.ModuleGather(useExtended=False, autoWalk=True)¶

Utility class for gathering and importing all modules in the music21 package. Puts them in self.modulePaths.

>>> from music21.test import commonTest
>>> mg = commonTest.ModuleGather(useExtended=True)
>>> print mg.modulePaths[0]
D:\Web\eclipse\music21base\music21\volume.py
